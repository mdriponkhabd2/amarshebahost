# **App Name**: AmarShebaHost

## Core Features:

- Hero Section Display: Showcases the brand's primary value proposition and call to action 'Get Started' with a professional, blue-themed background.
- Features Showcase: Displays key hosting features using clear cards and icons, such as High Speed Servers, Uptime Guarantee, and Customer Support.
- Dynamic Pricing Plans: Presents multiple hosting plans (Basic, Standard, Premium) with detailed features, pricing, and a 'Buy Now' button, highlighting the most popular option.
- Domain Search Tool: Provides an input field for users to search for domain name availability, powered by an external domain registration service API.
- About Section: Provides a concise description of AmarShebaHost, emphasizing reliability and local customer support.
- Customer Testimonials: Displays customer reviews and ratings to build trust and credibility.
- Site Navigation and Footer: Includes essential links (Home, Pricing, Contact, Support) and social media icons for easy site navigation and brand engagement.

## Style Guidelines:

- Primary Color: A vibrant, professional blue (#1466B8). This color communicates trust and technological advancement, anchoring the brand's identity as reliable and modern.
- Background Color: A very light, cool grey-blue (#F0F2F4). This choice creates a clean and expansive canvas, enhancing readability and maintaining a sophisticated, minimal aesthetic.
- Accent Color: A deep, rich indigo/violet (#2E177A). This accent provides strong contrast and depth against the primary and background, signaling premium quality and distinction without veering into teal.
- Body and headline font: 'Poppins', a geometric sans-serif for a precise, contemporary, and easily readable appearance, suitable for a modern web hosting service.
- Modern icon library: Utilize a library of clean, minimalist, and vectorized icons that align with a high-tech and professional brand image.
- Responsive Design: The layout is fully adaptable across mobile, tablet, and desktop devices, ensuring optimal user experience on any screen size. Elements feature soft shadows and rounded corners to convey a friendly and contemporary feel.
- Subtle Animations: Implement smooth transitions for element loading (fade-in, slide-up) and engaging hover effects on interactive components like buttons and cards, enhancing the user experience without distraction.
